# CVGPOST — Video Production Portfolio & Booking Platform

## 🚀 Live Demo
https://cvgpost.vercel.app

## 📌 Overview
CVGPOST is a cinematic, mobile-first portfolio and booking platform built for a video production startup. It allows users to explore services, view pricing, and book projects seamlessly.

## 🛠 Tech Stack
- Frontend: React.js, Tailwind CSS
- Backend: Node.js, Express.js
- Database: MongoDB
- Animations: Framer Motion
- Deployment: Vercel

## ✨ Features
- Cinematic and responsive UI
- Service showcase with pricing carousel
- Dynamic booking system with validation
- Smooth animations for better UX
- Optimized performance
